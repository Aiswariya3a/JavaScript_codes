A switch statement gives a more descriptive way to compare a value with multiple variants. It can replace multiple if checks.

The switch has one or more case blocks and an optional default.

Syntax


switch(x) {
case 'value1': // if (x === 'value1')
...
[break]

case 'value2': // if (x === 'value2')
...
[break]

default:
...
[break]
}

  • The value of x is checked for a strict equality to the value from the first case and so on.
  • If the equality is found, switch starts to execute the code starting from the corresponding case, until the nearest break (or until the end of switch).
  • If no case is matched then the default code is executed (if it exists).

If there is no break then the execution continues with the next case without any checks.
Several variants of case which share the same code can be grouped.
