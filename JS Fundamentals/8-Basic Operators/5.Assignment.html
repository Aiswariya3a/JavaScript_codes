when we assign a variable, like x = 2 * 2 + 1, the calculations are done first and then the = is evaluated, storing the result in x.
'=' is an operator - All operators in JavaScript return a value - '=' returns a value too.
The call x = value writes the value into x and then returns it.

Chaining assignments

a = b = c = 2 + 2 is same as
c = 2 + 2;
b = c;
a = c;
Chained assignments evaluate from right to left.
