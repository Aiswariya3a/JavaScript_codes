binary + is applied to strings, it merges (concatenates) Strings.
if any of the operands is a string, then the other one is converted to a string too.

alert(2 + 2 + '1' ) - returns "41" and not "221"
operators work one after another. The first + sums two numbers, so it returns 4, then the next + adds the string 1 to it, so it is like 4 + '1' = '41'.
alert('1' + 2 + 2) - returns "122" and not "14"
the first operand is a string, the compiler treats the other two operands as strings too.
The binary + is the only operator that supports strings in such a way. Other arithmetic operators work only with numbers and always convert their operands to numbers.
