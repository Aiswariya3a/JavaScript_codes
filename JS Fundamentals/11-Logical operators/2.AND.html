The AND operator is represented with two ampersands &&
AND returns true if both operands are truthy and false otherwise

true && true => true
false && true => false
true && false => false
false && false => false

Given multiple AND'ed values, Evaluates operands from left to right.
For each operand, converts it to a boolean
If the result is false, stops and returns the original value of that operand.
If all were truthy, returns the last operand.
The precedence of AND && operator is higher than OR ||.
