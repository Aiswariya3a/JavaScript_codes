A label is an identifier with a colon before a loop (basically a name for the for-loop)
labelName: for (...) {
...
}
break <labelName> statement, the loop breaks the labeled loop
continue <labelName> statement, the code execution jumps to the next iteration of the labeled loop.

Labels do not allow us to jump into an arbitrary place in the code.
break label; // jump to the label below (doesn't work)
label: for (...)
