break


we can force the exit at any time using the special break directive.
The combination "infinite loop + break as needed" is great for situations when a loop's condition must be checked in the middle or even in several places of its body.

Continue


continue stops the current iteration and forces the loop to start a new one (if the condition allows).
both cannot be used with the ternary operator ?.This is just another reason not to use the question mark operator ? instead of if.
