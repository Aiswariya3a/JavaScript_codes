syntax


for (begin; condition; step) {
// ... loop body ...
}
begin executes once, and then it iterates: after each condition test, body and step are executed.
a loop exits when its condition becomes falsy.

The “counter” variable i can be declared right in the loop (let i=0;) This is called an “inline” variable declaration. Such variables are visible only inside the loop.
Any part of for can be skipped. (begin, step or condition)
for(;;) => infinite loop
