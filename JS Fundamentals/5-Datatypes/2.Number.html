The number type represents both integer and floating point numbers.
Besides regular numbers, there are so-called “special numeric values” which also belong to this data type: Infinity, -Infinity and NaN.

NaN

NaN represents a computational error. It is a result of an incorrect or an undefined mathematical operation
NaN is sticky. Any further mathematical operation on NaN returns NaN
if there is a NaN somewhere in a mathematical expression, it propagates to the whole result (one exception to that: NaN ** 0 is 1).
The script will never stop with a fatal error (“die”). At worst, we will get NaN as the result.
